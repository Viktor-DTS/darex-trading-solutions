# NewServiceGidra - Оптимізована версія DTS

## Опис проекту

Цей проект є повністю переробленою версією системи DTS з наступними покращеннями:

### Основні зміни:

1. **Оптимізація запитів до бази даних:**
   - Прибрано обмеження `limit` - запити опитують всю базу
   - Складні агрегаційні запити MongoDB на рівні бекенду
   - Всі запити виконуються за < 300мс
   - Використання індексів та оптимізація запитів

2. **Новий дизайн:**
   - Повністю перероблений UI/UX
   - Сучасний та інтуїтивний інтерфейс
   - Адаптивний дизайн

3. **Покращена система колонок:**
   - Drag & drop для зміни порядку колонок
   - Збереження налаштувань колонок
   - Гнучке налаштування відображення

4. **Збережена логіка:**
   - Вся бізнес-логіка розрахунків
   - Логіка відображення заявок
   - Логіка залежності виконаних робіт
   - Вся логіка по вкладках та панелях

## Структура проекту

```
NewServiceGidra/
├── backend/          # Node.js + Express + MongoDB
├── frontend/         # React + Vite
├── README.md         # Цей файл
└── .gitignore
```

## Встановлення

### Backend:
```bash
cd backend
npm install
npm start
```

### Frontend:
```bash
cd frontend
npm install
npm run dev
```

## Технології

- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Frontend:** React, Vite, CSS Modules
- **База даних:** MongoDB

## Особливості оптимізації

- Агрегаційні пайплайни MongoDB для складних запитів
- Паралельне виконання запитів через Promise.all
- Використання lean() для швидших запитів
- Індекси на критичних полях
- Кешування на рівні бекенду
